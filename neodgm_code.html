Neo둥근모 Code

Neo둥근모를 기반으로 만들어진 Neo둥근모 Code 글꼴은 OpenType 고급 타이포그래피 기능을 이용하여 두 개 이상의 문자로 이루어진 기호를 하나로 합쳐서 표시함으로써 프로그램 코드 작성을 즐겁게 해 줍니다.

참고

  • Neo둥근모 Code는 'calt' OpenType 기능을 이용하여 ligature를 구현합니다. 응용 프로그램이 ligature 출력을 지원하는 경우, 설정을 통해 ligature 문자를 사용할 수 있습니다. CSS를 직접 사용해야 하는 경우에는 font-feature-settings: 'calt' on 선언을 추가하세요.
  • 응용 프로그램에서 ligature를 켜거나 끌 수 있는 옵션을 제공하는 경우에는 Neo둥근모 Code 글꼴만 설치해도 충분합니다. Ligature 기능을 끈 경우에는 Neo둥근모와 동일하게 문자가 표시됩니다.
