소개

Neo둥근모는 90년대에 퍼블릭 도메인으로 공개된 비트맵 한글 글꼴 둥근모꼴을 트루타입 형식으로 변환하고, 여기에 원래 글꼴에는 없던 여러가지 기호를 추가함으로써 여러 환경에서 유용하게 쓰일 수 있는 글꼴을 만드는 프로젝트입니다.

Neo둥근모 내려받기
v1.50 (GitHub)

특징

새로운 영문/숫자 자형

원래 둥근모꼴에는 함께 사용되도록 만들어진 영문 비트맵 글꼴도 포함되어 있습니다. 하지만 그 글꼴은 여러모로 자형이 미려하지 않기 때문에 둥근모꼴의 한글 글자 모양과 어울리는 비트맵 폰트를 나름대로 만들어 넣었습니다. 둥근모꼴 특유의 모서리가 둥근 산세리프 스타일을 최대한 표현하는 것에 중점을 두었습니다.

Unicode에 등록된 다양한 문자 제공

Neo둥근모의 목표는 한글과 영문/숫자뿐만 아니라 여러가지 Unicode 문자도 지원하는 것입니다. 이와 더불어, 터미널의 셸 프롬프트나 Vim 등의 텍스트 편집기, 또는 Tmux 등의 환경에서 사용 가능한 Powerline 특수기호도 Neo둥근모의 스타일에 맞춰서 제공하고 있습니다. 현재 지원하는 문자들은 아래에서 확인하실 수 있습니다.

프로그램 코드용 Ligature 제공 (Neo둥근모 Code)

Neo둥근모를 기반으로 만들어진 Neo둥근모 Code 글꼴은 OpenType 고급 타이포그래피 기능을 이용하여 두 개 이상의 문자로 이루어진 기호를 하나로 합쳐서 표시할 수 있습니다. 현재 ligature를 지원하는 기호들은 여기에서 확인하실 수 있습니다.

Neo둥근모 미리보기

  • 아래 버튼을 클릭하면 해당 범주에 속하는 문자들을 펼쳐볼 수 있습니다.
  • 문자의 썸네일에 마우스 포인터를 올리면 Unicode 코드 포인트가 표시됩니다.
